Java实现歌词文本格式的微调

最近 QQ 音乐的会员到期了，反正鼠鼠是不想再续费了，偶然间发现一个网站可以爬到歌曲歌词信息的文件，但是格式不是我想要的，于是写个程序帮我改格式。

处理前：LRC1.txt

处理后：LRC2.txt

很容易能看出来区别，就不阐述格式有什么变化了。

然后也没有什么特殊的写法，总结一下步骤：

  1. 读取源文件

  2. 逐行读取文件内容并添加到StringBuilder中

  3. 将StringBuilder转换为字符串，并按换行符分割成数组

  4. 用一个BufferedWriter来写进LRC2.txt

  5. 遍历这个line数组

  6. 二次分割

  7. 直接把l[0]写进去（l[0]是[时间]外语 翻译）

  8. 把时间取出来，再把l[1]加上

这样就完成了，具体代码详见：Main.java
