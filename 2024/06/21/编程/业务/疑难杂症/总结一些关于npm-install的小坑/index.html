总结一些关于npm install的小坑

npm真的坑，在我初学的时候真的有很多坑，学起来真的累，但好在吃一堑长一智，慢慢摸透，这就是所谓孰能生巧吧。

特别是，我前端不是那么擅长，基本都是参考网上的demo写法，因此常常使用npm install命令，这个命令老烦了因此我打算把遇到的坑都记录下来。

异常1

npm ERR! code EINTEGRITY npm ERR! sha1-xxxxxx integrity checksum failed when using sha1: wanted sha1-xxxxxx but got sha512-xxxxxx sha1-yyyyyyy. (10379 bytes)

解决方式：

直接删除 package-lock.json 文件，重新执行 npm install 即可
