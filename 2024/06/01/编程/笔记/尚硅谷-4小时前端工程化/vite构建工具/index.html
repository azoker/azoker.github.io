vite构建工具

vite 是 vue 项目构建的脚手架，也就是构建工具，它可以帮我们初始化一个 vue 项目，这意味咱们只有专注代码开发，省去了很多繁琐的配置。笔者在这里提供一个文档：Vite中文网

vite 构建项目

没有安装 vite 会提示你安装，直接回车就好。

接下来按照他的命令一步一步来：

vite 打包

用这个命令打包之后在项目根目录会生成一个 dist 文件夹，可以把这个 dist 文件夹的所有文件放在服务器上（例如 nginx ）这样就打包好了。当然，交给 tomcat 去管理也可以，即放在 Spring Boot 项目的 resource 文件夹下。

vue 项目目录

public目录

这个是可以直接浏览器访问的，比如新建一个hello.json文件，数据是{"username":"zhangsan"}，然后启动项目，直接访问localhost:5173/hello.json，就能看到数据了。

src目录

放源码的，一般代码开发就是在这个目录做的。
